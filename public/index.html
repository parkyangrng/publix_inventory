Endpoints

Every endpoint accepts POST with Content-Type: application/json

POST
/api/inventory
Browse stock levels across all stores
{ store_id, category, in_stock, page, limit }
POST
/api/inventory/sku
Stock for one SKU across all or one store
{ sku*, store_id }
POST
/api/inventory/low-stock
Items at or below restock threshold (≤20 units)
{ store_id, category }
POST
/api/inventory/search
Full-text search by name, SKU, brand, description
{ q*, store_id, category, in_stock }
POST
/api/products
Full product catalog with optional filters
{ category, brand, organic, page, limit }
POST
/api/products/detail
Full detail record for a single product
{ sku* }
POST
/api/products/availability
Product detail + live stock in one call
{ sku*, store_id }
POST
/api/stores
All active Publix store locations
{ active_only }
POST
/api/stores/summary
Store-level inventory snapshot
{ store_id* }
POST
/api/categories
All categories with product counts
{ slug }
POST
/api/categories/inventory
Stock overview for a category
{ category*, store_id, in_stock }

Example requests

curl examples -X POST
Browse produce in stock
curl -X POST https://publix-inventory-api.onrender.com/api/inventory \
  -H "Content-Type: application/json" \
  -d '{"category":"produce","store_id":"store_FL01","in_stock":true}'
Search inventory
curl -X POST https://publix-inventory-api.onrender.com/api/inventory/search \
  -H "Content-Type: application/json" \
  -d '{"q":"oat milk","category":"dairy"}'
Product + live stock
curl -X POST https://publix-inventory-api.onrender.com/api/products/availability \
  -H "Content-Type: application/json" \
  -d '{"sku":"PRD-BAN-ORG"}'
Low stock at store
curl -X POST https://publix-inventory-api.onrender.com/api/inventory/low-stock \
  -H "Content-Type: application/json" \
  -d '{"store_id":"store_FL01"}'
Store snapshot
curl -X POST https://publix-inventory-api.onrender.com/api/stores/summary \
  -H "Content-Type: application/json" \
  -d '{"store_id":"store_FL02"}'
Category inventory
curl -X POST https://publix-inventory-api.onrender.com/api/categories/inventory \
  -H "Content-Type: application/json" \
  -d '{"category":"dairy","in_stock":true}'
